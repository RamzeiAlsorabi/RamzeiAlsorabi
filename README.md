### 🚀 Strategic Projects

🌐 **Smart Campus Network Management System** *(Flagship Project)*

نظام متكامل لمراقبة وإدارة البنية التحتية للشبكة في الحرم الجامعي، باستخدام تقنيات مراقبة الشبكة (SNMP، NetFlow)، والأتمتة (Ansible، Python)، وتحليل البيانات (Elastic Stack)، مع لوحة تحكم تفاعلية تعرض حالة الشبكة، أداء الأجهزة، واكتشاف التهديدات اللحظية.

**الميزات**:
- مراقبة حية لأجهزة الشبكة (Switches, Routers, Firewalls) عبر SNMP.
- تحليل تدفق البيانات (NetFlow/sFlow) لكشف الحالات الشاذة والاختناقات.
- نظام إنذار ذكي (Alerting) عند انقطاع الخدمات أو الهجمات.
- أتمتة مهام الصيانة (تحديث الأجهزة، نسخ احتياطي للتكوينات) باستخدام Ansible.
- لوحة تحكم (Dashboard) بإحصائيات الأداء والاستخدام باستخدام Grafana.

**التقنيات المستخدمة**:
- **Backend**: Python (Django/Flask) مع مكتبات SNMP و NetFlow.
- **Frontend**: React.js / Grafana للوحات التحكم.
- **Database**: InfluxDB (زمني)، PostgreSQL.
- **Automation**: Ansible, Python Scripts.
- **Monitoring**: Nagios, Zabbix, ELK Stack.
